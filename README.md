# 使用vue.js高仿网易云音乐
####基于学习为目的
## 技术栈
```
1.vue vue-router vuex
2.axios
3.Muse-UI
```
## 安装运行
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
